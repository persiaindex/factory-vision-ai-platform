# Final Demo Script

## Demo goal

Show that a dropped image moves through the full system automatically.

## Steps

1. Start Django
2. Start FastAPI
3. Start the watcher
4. Start React
5. Open the React dashboard
6. Drop a sample image into `shared/input/`
7. Watch the dashboard update automatically
8. Explain the displayed image, bounding box, detections, and job history

## Key sentence to say

A new image dropped into the input folder is automatically uploaded, processed, stored, and shown in the dashboard without manual refresh.
