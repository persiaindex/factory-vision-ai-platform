# Calibration Notes

## Current approach

Version 1 uses a fixed millimeters-per-pixel scale for a single camera setup.

## Assumptions

- camera position is stable
- object distance is stable enough for version 1
- bounding box size is acceptable as a rough proxy for object size

## Limitation

This is simpler than full camera calibration and perspective-aware measurement.

## Why it is acceptable

It is enough to demonstrate industrial post-processing logic in a portfolio project.
